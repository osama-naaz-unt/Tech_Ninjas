Start a terminal / command prompt window in this directory i.e. root directory, inside the E-Commerce folder that has this file

Ensure you have Python 3 installed on your computer. You can confirm by running the following command:
>> python --version

Install the required libraries i.e. Django (and Faker for generating sample fake data) by running the following command:
>> pip install -r requirements.txt

Start the server by running the following command:
>> python manage.py runserver

Access the website under development by opening the shown link in your browser i.e. 127.0.0.1:8000
